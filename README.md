# Boolean Construction Template

**Generate precision-tuned Boolean search strings for frontier AI roles.**

Just provide a JD—Claude extracts, analyzes, and outputs a styled, interactive HTML file with copy buttons.

## What It Does

- Extracts tools, benchmarks, frameworks from job descriptions
- Organizes Boolean logic into Blocks → Sub-blocks → Clusters
- Generates collapsible, scannable HTML output
- Includes ready-to-use platform strings (LinkedIn, GitHub, X)
- Infers candidate archetypes with search paths

## How to Use

1. Download `Boolean_Construction_Template_v4.8.md`
2. Attach to Claude (Opus 4.5 recommended)
3. Paste your job description
4. Claude generates a standalone HTML file
5. Open in browser — fully interactive
